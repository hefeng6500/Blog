## 面向对象

面向对象编程：是一种编程范式或者编程风格、以类或者对象作为组织代码的基本单元，并将封装、抽象、继承、多态作为代码设计和实现的基石。

面向对象编程语言：支持类和对象的语法机制，并有现成的语法机制实现面向对象编程四大特性的编程语言。

**面向对象的四大特性**

封装、抽象、继承、多态
